### StandardDeviation

Calculates the standard deviation in a sliding window of the input time series.

**Inputs**
**in**
Input time series


**Outputs**
**out**
Standard deviation


**Parameters**
**minSamples**
Minimum number of observations for producing output
**windowLength**
Length of the sliding window (number of observations)
