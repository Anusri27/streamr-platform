### UnivariateLinearRegression

Performs a least-squares linear regression on a sliding window of input data. The model is <span class="highlight">y = slope*X + intercept.</span>

**Inputs**
**inX**
Input X values
**inY**
Input Y values


**Outputs**
**error**
Mean square error (MSE) of the fit
**intercept**
Intercept of the linear fit
**slope**
Slope of the linear fit
**R^2**
R-squared value of the fit


**Parameters**
**windowLength**
Length of the sliding window as number of samples
