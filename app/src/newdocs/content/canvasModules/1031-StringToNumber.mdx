### StringToNumber

Parse a number from the input string.  

Examples of valid floating-point numbers:  


- "1"- "3.14159"- "-.234e4" (outputs -2340)- "+3.e1" (outputs 30)

**Inputs**
**in**
input string


**Outputs**
**out**
number parsed from input string
**error**
Error if input can't be parsed
