### GetMultiFromMap

Get multiple values from a Map.  **The names of outputs are used as map keys so make sure to change them!**

**Inputs**
**in**
input map


**Outputs**
**founds**
an array indicating for each output with 0 (false) and (1) whether a value was found
**out-1**
a (default) value from map, output name is used as key
