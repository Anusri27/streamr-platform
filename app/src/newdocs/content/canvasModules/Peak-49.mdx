Attempts to detect upward turns below <span class="highlight">lowZone</span> (outputs 1) and downward turns below <span class="highlight">highZone</span> (outputs -1).
  


For an upward turn to be registered, the change between subsequent input values must be larger than <span class="highlight">threshold</span>. For a downward turn the change must be less than <span class="highlight">-threshold</span>.
