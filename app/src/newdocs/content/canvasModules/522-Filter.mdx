### Filter

Only lets the incoming value through if the value at <span class="highlight">pass</span> is 1. If this condition is not met, no event is sent out.

**Inputs**
**pass**
The filter condition. 1 (true) for letting the event pass, 0 (false) to filter it out
**in**
The incoming event (any type)


**Outputs**
**out**
The event that came in, if passed. If filtered, nothing is sent
