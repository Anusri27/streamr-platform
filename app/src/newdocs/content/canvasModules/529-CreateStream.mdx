### CreateStream

Create a new stream.

**Inputs**
**name**
name of the stream
**description**
human-readable description


**Outputs**
**created**
true if stream was created, false if failed to create stream
**stream**
the id of the created stream


**Parameters**
**fields**
the fields to be assigned to the stream
