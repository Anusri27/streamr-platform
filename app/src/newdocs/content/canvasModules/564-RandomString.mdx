### RandomString

Generate fixed-length random strings from an equiprobable symbol pool. Allowed symbols can be configured from module settings.

**Inputs**
**trigger**
when value is received, activates module


**Outputs**
**out**
the random string


**Parameters**
**length**
length of strings to generate
