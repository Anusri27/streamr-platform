### Sum

Calculates the (optionally rolling) sum of incoming values. For an infinite sum, enter a <span class="highlight">windowLength</span> of 0.

**Inputs**
**in**
Values to be summed


**Outputs**
**out**
Sum of values in the window


**Parameters**
**minSamples**
How many values must exist in the window before outputting a value
**windowLength**
Length of the sliding window of values to be summed, or 0 for infinite
