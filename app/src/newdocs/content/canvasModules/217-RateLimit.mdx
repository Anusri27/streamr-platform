### RateLimit

The RateLimit module lets through n messages in t milliseconds. Then module just blocks the rest which do not fit in the window.

**Inputs**
**in**
Input


**Outputs**
**limitExceeded?**
Outputs 1 if the message was blocked and 0 if it wasn't
**out**
Outputs the input value if it wasn't blocked


**Parameters**
**rate**
How many messages are let through in given time
**timeInMillis**
The time in milliseconds, in which the given number of messages are let through
