### NewMap

Emit a map everytime trigger receives a value.

**Inputs**
**trigger**
used to activate module


**Outputs**
**out**
a map


**Parameters**
**alwaysNew**
When false (defult), same map is sent every time. When true, a new map is sent on each activation.
