### SumByKey

Keeps aggregated sums of received key-value-pairs by key.

**Inputs**
**value**
The value to be added to aggregated sum.
**key**
The (string) key


**Outputs**
**map**
Key-sum pairs
**valueOfCurrentKey**
The aggregated sum of the last key received. 


**Parameters**
**windowLength**
Limit moving window size of sum.
**sort**
Whether key-sum pairs should be order by sums
**maxKeyCount**
Maximum number of (sorted) key-sum pairs to keep. Everything else will be dropped.
