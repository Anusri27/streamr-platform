### SQL

The result is a list of map objects, e.g. *[{"id":0, "name":"Me"}, {"id":1, "name":"You"}]*

**Inputs**
**sql**
SQL command to be executed


**Outputs**
**errors**
List of error strings
**result**
List of rows returned by the database


**Parameters**
**engine**
Database engine, e.g. MySQL
**host**
Database server to connect
**database**
Name of the database
**username**
Login username
**password**
Login password
